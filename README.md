
#APP Noticias  📰

-------------

Aplicación para la visualización de noticias de diversas páginas.
## Instrucciones 👀

- Crear una cuenta con nombre, correo y contraseña (min 6 carácteres).
- Iniciar Sesión
- A disfrutar noticias 😚!
